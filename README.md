# PulseOS — AI Life Assistant

PulseOS is a full-stack AI-powered productivity and life management system designed to evolve into a true AI life assistant.

PulseOS is an AI-powered life assistant that helps users manage tasks, plan their day, organize information, and boost productivity through smart automation and conversational support.

It combines task management, goal tracking, journaling, and AI-powered insights into a unified intelligent system.

---

## 🚀 Features

### 📊 Dashboard
- Productivity momentum score
- Smart insights and recommendations
- Activity overview

### ✅ Tasks System
- Kanban-style task board
- Priority management
- Filtering and search
- Focus tracking

### 🎯 Goals System
- Progress tracking
- Milestones
- Analytics dashboard
- Momentum visualization

### 📓 Journal
- Mood tracking
- Reflection entries
- AI reflection prompts
- Emotional insights

### 🤖 AI Chat Assistant
- Context-aware AI responses
- Planning and summarization
- Goal and task assistance
- Productivity coaching

## 🧠 AI Vision

PulseOS is evolving into a true AI Life Assistant that will:

- Analyze goals and progress

## 🛠 Tech Stack

### Frontend
- React + Vite
- TailwindCSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- Prisma ORM
- Supabase PostgreSQL
- JWT Authentication

### AI Layer
- Groq API
- LLaMA 3.1 8B Instant Model
